Portfolio

Qu'est-ce qu'un portfolio ?

Un portfolio en informatique, est en quelque sorte un dossier personnel où sont regroupées les diverses compétences d'un développeur et permet aux employeurs de jauger les capacités techniques de celui-ci.

Vous trouverez dans cette section l'ensemble des projets sur lesquels j'ai pu travailler.
Cela vous permettra d'avoir un avis plus précis sur mes compétences techniques ainsi que la possibilité d'avoir accès au code de certains de mes projets via les repos sur Github.
